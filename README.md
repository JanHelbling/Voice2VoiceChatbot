# Voice2VoiceChatbot - Update von Jan Helbling

### Installation
1. `conda create -n test`
2. `conda activate test`
3. `pip install -r requirements.txt`
4. `python script.py`

Oder ohne Conda:

1. `pip install -r requirements.txt`
2. `python script.py`
